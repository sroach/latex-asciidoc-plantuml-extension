== Demo 3

=== Ascii Doc Source Code
[source,asciidoc]
....
[latex,demo3]
----
@startmath
2x^2 + 3(x-1)(x-2)  = 2x^2 + 3(x^2-3x+2) = 2x^2 + 3x^2 - 9x + 6 = 5x^2 - 9x + 6
@endmath
----
....

=== Output

[latex,demo3]
----
@startmath
2x^2 + 3(x-1)(x-2)  = 2x^2 + 3(x^2-3x+2) = 2x^2 + 3x^2 - 9x + 6 = 5x^2 - 9x + 6
@endmath
----
